# DEVOPS P3 VAGRANT

## Installation

MacOS:

```sh
$ brew cask install virtualbox
$ brew cask install vagrant
```

## Usage
Website resources are automatically cloned from `https://github.com/adyassine/p3-devops-docker` into `p3-devops.com`
